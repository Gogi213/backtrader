# 🚀 KAMA + Adaptive Z-Score Strategy

## 📋 Описание

Новая стратегия специально разработанная для коротких таймфреймов (15s). Сочетает:
- **KAMA (Kaufman's Adaptive Moving Average)** - адаптивный трендовый фильтр
- **Адаптивный Z-Score** - mean reversion сигналы с динамическим окном

## 🎯 Логика стратегии

### **Идея:**
Покупать/продавать против краткосрочных экстремумов в направлении долгосрочного тренда.

### **Условия входа:**

**Long (покупка):**
- KAMA направлен вверх (восходящий тренд)
- Адаптивный Z-Score ≤ -threshold (цена сильно упала от краткосрочного среднего)

**Short (продажа):**
- KAMA направлен вниз (нисходящий тренд)
- Адаптивный Z-Score ≥ +threshold (цена сильно выросла от краткосрочного среднего)

## 🔧 Параметры стратегии

### **KAMA параметры:**
- **KAMA Period** (20) - период для расчета эффективности
- **KAMA Fast SC** (2) - быстрая константа сглаживания
- **KAMA Slow SC** (30) - медленная константа сглаживания

### **Адаптивный Z-Score:**
- **Z-Score Base Window** (10) - базовое окно для Z-Score
- **Z-Score nATR Multiplier** (50) - множитель nATR для адаптации
- **Z-Score Threshold** (2.0) - порог для входа

### **Адаптивность окна:**
```
Окно Z-Score = Base Window + nATR% × Multiplier

Примеры:
- Низкая волатильность (nATR=0.5%): Окно = 10 + 0.5×50 = 35
- Высокая волатильность (nATR=2.0%): Окно = 10 + 2.0×50 = 110
```

## 🎛️ Стандартные параметры

- **Stop Loss (%):** 1.0%
- **Take Profit (%):** 3.0% 
- **Use nATR Filter:** false
- **Min nATR (%):** 0.5%

## 💡 Преимущества для 15s

1. **KAMA адаптивность:** Быстро реагирует на изменения, фильтрует шум
2. **Динамическое окно:** Больше волатильность → больше окно → меньше ложных сигналов
3. **Mean reversion:** Хорошо работает на коротких интервалах
4. **Трендовый фильтр:** Входы только в направлении основного тренда

## 📈 Пример работы

```
Ситуация: Восходящий тренд, временная коррекция
1. KAMA ↗ (тренд вверх)
2. Цена упала, Z-Score = -2.5 (oversold) 
3. → LONG сигнал
4. Цена отскакивает к среднему → прибыль
```

## ⚙️ Как запустить

### В UI:
1. Выберите стратегию: "KAMA + Adaptive Z-Score"
2. Настройте параметры для вашего актива
3. Для 15s рекомендуется:
   - KAMA Period: 15-25
   - Z-Score Base Window: 8-12
   - Z-Score Threshold: 1.5-2.5

### Программно:
```python
from panel_app.strategies.kama_zscore import run_kama_zscore_strategy

pf, signals = run_kama_zscore_strategy(
    df,
    kama_period=20,
    z_base_window=10,
    z_threshold=2.0
)
```

## 🔬 Рекомендации по настройке

### **Для разной волатильности:**
- **Низкая волатильность:** Уменьшить threshold (1.5), увеличить multiplier (70)
- **Высокая волатильность:** Увеличить threshold (2.5), уменьшить multiplier (30)

### **Для разных активов:**
- **Форекс:** KAMA Period 15-20, быстрые параметры
- **Крипто:** KAMA Period 20-30, более консервативные настройки
- **Акции:** Стандартные параметры
